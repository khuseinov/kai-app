03.2 — kai_theme.dart

Тема.

KaiTheme.light() и KaiTheme.dark() возвращают полный ThemeData. Подключите один раз в MaterialApp — каждый виджет ниже подхватит цвета автоматически.

lib/core/theme/kai_theme.dartdart
import 'package:flutter/material.dart'; import 'package:google_fonts/google_fonts.dart'; class KaiTokens { // Градиент прилива — независимый от темы static const tideStop1 = Color(0xFF1B4FB0); static const tideStop2 = Color(0xFF2BA8C9); static const tideStop3 = Color(0xFFF4B589); static final tideGradient = LinearGradient( begin: Alignment.topLeft, end: Alignment.bottomRight, colors: [tideStop1, tideStop2, tideStop3], stops: [0.0, 0.52, 1.0], ); } class KaiTheme { static ThemeData light() => ThemeData( brightness: Brightness.light, scaffoldBackgroundColor: const Color(0xFFFAFAF9), colorScheme: const ColorScheme.light( primary: Color(0xFF2C5BE5), surface: Color(0xFFFFFFFF), onSurface: Color(0xFF111114), error: Color(0xFFC44A3C), ), textTheme: GoogleFonts.manropeTextTheme(), extensions: [_kaiTokensExt(light: true)], ); static ThemeData dark() => ThemeData( brightness: Brightness.dark, scaffoldBackgroundColor: const Color(0xFF0E0E11), colorScheme: const ColorScheme.dark( primary: Color(0xFF5C8EFF), surface: Color(0xFF16161A), onSurface: Color(0xFFF5F5F2), error: Color(0xFFE66F60), ), textTheme: GoogleFonts.manropeTextTheme(ThemeData.dark().textTheme), extensions: [_kaiTokensExt(light: false)], ); }
03.3 — KaiTideCurve

Знак бренда в коде.

Один виджет управляет всеми восемью состояниями. Передайте enum — получите нужную анимацию. CustomPainter + AnimationController. Без сторонних библиотек анимации.

lib/widgets/kai_tide_curve.dartdart
enum TideState { idle, listening, thinking, responding, success, error, memory, sleep, } class KaiTideCurve extends StatefulWidget { final TideState state; final double height; final EdgeInsets padding; const KaiTideCurve({ super.key, required this.state, this.height = 16, this.padding = const EdgeInsets.symmetric(horizontal: 18), }); // Профиль анимации для каждого состояния — см. design-tokens.json § tide-states Duration get _duration => switch (state) { TideState.listening => Duration(milliseconds: 2200), TideState.thinking => Duration(milliseconds: 3000), TideState.responding => Duration(milliseconds: 1400), TideState.success => Duration(milliseconds: 1200), TideState.error => Duration(milliseconds: 600), TideState.memory => Duration(milliseconds: 900), _ => Duration.zero, }; // reduce-motion → свернуть анимации до статичной прозрачности // Читать MediaQuery.disableAnimationsOf(context) внутри build() }
03.4 — виджеты

Ещё три виджета.

Кнопка, пузырь сообщения, поле ввода. Все используют тему. Кнопка с заливкой Прилива — момент бренда, одна на экран.

lib/widgets/kai_button.dartdart
enum KaiButtonVariant { primary, tide, secondary, ghost, destructive } class KaiButton extends StatelessWidget { final String label; final VoidCallback? onPressed; final KaiButtonVariant variant; final Widget? icon; final bool isLoading; const KaiButton({ super.key, required this.label, this.onPressed, this.variant = KaiButtonVariant.primary, this.icon, this.isLoading = false, }); // Радиус: 14, отступы: 12×18, вес: 600, трекинг: -0.005em // вариант .tide использует KaiTokens.tideGradient как фон — анимирован при наведении } class KaiBubble extends StatelessWidget { final String text; final bool isUser; final List<Source> sources; final bool streaming; // Пользователь: фон surface-2, радиус 14-14-4-14, по правому краю // Kai: без фона, глиф прилива над именем, встроенные цитаты [n] в акценте } class KaiCompose extends StatefulWidget { final ValueChanged<String> onSubmit; final VoidCallback? onMicTap; // Многострочный, макс 80px, кнопка отправки с градиентом прилива }
03.5 — анимация

Одна кривая, три длительности.

Один токен смягчения покрывает 90% случаев. Стандарт 240 мс для UI, фоновый 2.6 с для циклов бренда, микро 120 мс для обратной связи нажатия.

lib/core/motion/kai_motion.dartdart
class KaiMotion { // Длительности static const standard = Duration(milliseconds: 240); static const ambient = Duration(milliseconds: 2600); static const micro = Duration(milliseconds: 120); static const exitFast = Duration(milliseconds: 200); // Кривые смягчения static const standardCurve = Cubic(0.2, 0, 0, 1); // Material 3 выделенный static const ambientCurve = Curves.easeInOut; static const exitCurve = Cubic(0.4, 0, 0.6, 1); } // Пример использования — панель выезжает AnimatedPositioned( duration: KaiMotion.standard, curve: KaiMotion.standardCurve, left: panelOpen ? 0 : -panelWidth, child: KaiPanel(...), );
03.6 — быстрый старт

Три шага.

Добавить в kai-app, запустить, отправить.

01

Скопировать токены

Поместите design-tokens.json в lib/core/design/. Запустите flutter pub run build_runner build для генерации Dart-констант.

02

Подключить тему

В main.dart: MaterialApp(theme: KaiTheme.light(), darkTheme: KaiTheme.dark()).

03

Использовать виджеты

Замените существующие кнопки / пузыри / поле ввода на KaiButton, KaiBubble, KaiCompose. Добавьте KaiTideCurve вверху каждого экрана.
