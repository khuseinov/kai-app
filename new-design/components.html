03.8 — пузыри сообщений

Пузыри сообщений

03.12 — toast

Toast · Snackbar

Краткий системный фидбек: скопировано, сохранено, ошибка отправки, Kai запомнил. Pill-форм над compose island, 3 секунды, свайп для закрытия. Все варианты — тёмный фон (ink-1). Только «Kai запомнил» — tide gradient.

Позиция

Всегда над compose island — bottom: 58px. Центрован по горизонтали. Никогда не перекрывает текст в ленте. Slide-up on appear, fade-down on dismiss.

Время жизни

3 секунды — авто-dismiss. Memory toast показывает countdown bar. Негативный тост остаётся до действия пользователя (tap → dismiss или retry). Swipe вниз — мгновенное закрытие.

Правила

Только один toast одновременно. Новый вытесняет старый. Не использовать для важных алертов — только для эхо-фидбека действий. Важные события → Alert Card (N-01) в ленте.
