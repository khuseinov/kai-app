шаг 02 — набор бренда

Иконка, сплэш,
социальная карточка.

Готовые брендовые ресурсы. Иконка в трёх вариантах (основная / тёмная / моно) и уменьшенная до favicon. Сплэш для первого запуска. OG-карточка для шаринга ссылок. Краткий README бренда внизу — можно, нельзя, карта файлов.

02.1 — иконка приложения

Один знак, три контекста.

Кривая прилива внутри квадрата с радиусом 22% (iOS round-square). Основная = полный градиент прилива. Тёмная = глубокий тёмный фон с градиентной кривой. Моно = ink-1 с белой кривой — для монохромных трафаретов, циферблатов, тёмного сплэша.

основная

Градиент Прилива + белая кривая

Иконка по умолчанию. Основная для iOS / Android. Градиент полноблочный, кривая белая с обводкой 3px.
тёмная

Глубокий тёмный + кривая прилива

Для тонированной иконки тёмного режима iOS, тёмного сплэша, размещения бренда на тёмных поверхностях.
моно

Чернила + белая кривая

Циферблат, одноцветная печать, трафарет, режим высокого контраста для доступности.

02.3 — readme бренда

Как использовать знак.

Краткие правила. Покрывают 80% случаев. Для всего остального — по умолчанию кривая прилива у верхнего края, никогда не изобретайте.

делать

Сочетать иконку с логотипом

Набор логотипа — глиф кривой + логотип «kai» с отступом 14 px. Используйте на сплэше, заголовке сайта, маркетинговых материалах.

не делать

Перекрашивать цветовые точки градиента

Прилив заблокирован на #1B4FB0 → #2BA8C9 → #F4B589. Никаких «вариантов бренда», праздничных перекрасок, клиентских оттенков.

делать

Соблюдать радиус скругления 22% на иконке

Для всех квадратных поверхностей иконок (иконка приложения, глиф сплэша). На круглых аватарах используйте круг; кривая масштабируется.

не делать

Размещать иконку на загруженном фоне

Кривой нужен контраст. Используйте иконку на однотонных поверхностях (белая, чернила или один цвет Прилива). Никогда поверх фотографий или других градиентов.

делать

Использовать моно когда цветной непрактичен

Одноцветная печать, циферблат, стикер-бейдж, электронная бумага, режим доступности — рендерите моно-вариант.

не делать

Добавлять тень к иконке

Плоский и точный. iOS добавляет тень на уровне платформы; мы не удваиваем. То же на Android adaptive.
