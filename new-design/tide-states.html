04 · правила

Как пользоваться

Что нельзя нарушать при реализации состояний в продукте.

Переходы мгновенные

Смена состояния происходит на следующий фрейм — не плавно. Анимация внутри состояния плавная, переход между состояниями — нет. Easing только для duration, не для смены.

Эфемерные → возврат в idle

success, error, memory — короткие вспышки. После их завершения кривая автоматически возвращается в idle (или в предыдущее постоянное состояние если разговор продолжается).

Gradient неизменен

Tide gradient (#1B4FB0 → #2BA8C9 → #F4B589) не меняется ни при каком состоянии, кроме success (зелёный) и error (коралловый). Holiday recolors, тематические варианты — запрещены.
